Connect to the Natural World

Connect plants to the web using Arduino, Processing and Google Docs.

I have always been frustrated with the technical and time requirements needed to view physical sensor data on the web. Services like Xively, ThingSpeak and Nimbits provide excellent features they all require some prerequisite knowledge of enough web technologies that may come across as intimidating, especially to those who may be new to programing. While searching for a simpler method to teach to youth, I came across some wonderful examples on journal.code4lib.org and Instructables on how to utilize Google Docs as an easier solution.

Hardware used in this tutorial:
  • Temperature and Humidity Sensor (Grove)
  • Moisture Sensor (Grove)
  • Arduino Uno
  • Breadboard
Software used in this tutorial:
  • Arduino
  • Humidity Temperature Sensor - Arduino Library (How to install a library for Arduino)
  • Processing
  • HTTProcessing - Processing Library (How to install a library for Processing)

1. Arduino

  • Setup your sensors by assembling the breadboard as shown here: note: we are only using the power and ground rails on the breadboard

  • Open and Run Plant_Sensors.ino You should now see 3 variables streaming at the bottom of your serial window.

To get a better idea of what you are viewing, here are the value ranges for each sensor:

  • Humidity range = 20% - 90% RH
  • Temperature range = 0 - 50 C
  • Moisture range = 0 - 950

I designed this tutorial to give a brief overview of multiple software and hardware components.
